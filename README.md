# LoanLens - Credit Risk Assessment System

Machine learning-based loan approval prediction system built using Python and Scikit-learn.

## Features

- Data preprocessing
- Exploratory Data Analysis (EDA)
- Feature Engineering
- Logistic Regression Model
- Loan Approval Prediction

## Tech Stack

- Python
- Pandas
- NumPy
- Scikit-Learn
- Streamlit

## Model Performance

| Metric | Score |
|----------|----------|
| Accuracy | 87.5% |
| Precision | 79.0% |
| Recall | 80.3% |
| F1 Score | 79.7% |
